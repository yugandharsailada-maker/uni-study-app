# 🌌 Cortex

> The Academic Operating System.

Cortex is an AI-native academic workspace built for students who operate better with systems than schedules.

It combines semester management, predictive grade simulation, syllabus intelligence, and workflow tooling into a single operational environment focused on clarity, speed, and cognitive reduction.

Instead of behaving like another checklist application, Cortex functions more like a command center:
- semesters become datasets
- subjects become systems
- grades become simulations
- study becomes operational

Built on the **Lumiere Design Language**, Cortex blends glassmorphism, high-density layouts, and keyboard-first interaction into an interface that feels closer to a futuristic workstation than a traditional study app.

---

## ✦ What Cortex Does

### 🧠 AI Syllabus Intelligence

Paste a raw syllabus, module outline, or academic notice.

Using Gemini, Cortex transforms unstructured text into:
- semesters
- subjects
- assignment structures
- exam timelines
- academic workflows

What normally takes an hour becomes a few seconds.

---

### 📈 Predictive Grade Simulation

The Cortex Simulation Engine allows you to model academic outcomes before they happen.

Test:
- upcoming exam scenarios
- assignment weight adjustments
- GPA targets
- semester projections

Weighted evaluations and projected letter grades are recalculated dynamically in real time.

---

### 🍱 High-Density Dashboarding

Cortex is designed around glanceability and information compression.

Track:
- deadlines
- assignments
- exams
- semester metrics
- study priorities
- performance analytics

The interface prioritizes density without visual overload.

---

### ⌨️ Keyboard-First Workflow

Inspired by modern developer tooling.

Cortex minimizes friction through:
- Command Palette (`Ctrl + K`)
- global shortcuts
- rapid navigation
- modal-driven workflows
- instant semester management

The goal is operational flow — not menu traversal.

---

## ✦ Philosophy

Most academic software feels administrative.

Cortex is built on a different assumption:

> students are operators managing complex cognitive workloads.

The system focuses on:
- reducing mental fragmentation
- compressing academic overhead
- increasing situational awareness
- maintaining execution clarity

The interface stays lightweight so attention stays on work.

---

## ✦ Design Language — Lumiere

Lumiere is a glassmorphic interface system centered around:
- layered translucency
- atmospheric depth
- precision typography
- low-friction interaction
- architectural spacing

The aesthetic is intentionally restrained:
minimal noise, maximum focus.
